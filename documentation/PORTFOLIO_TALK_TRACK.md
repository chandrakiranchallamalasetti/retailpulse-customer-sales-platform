# Portfolio Talk Track

## Resume bullet examples
- Modernized a SQL Server customer-orders database into a deployment-ready analytics platform using SQLite, Python, pandas, Plotly, and Streamlit; implemented a five-table 3NF model, analytical views, interactive KPI dashboards, and customer/salesperson drilldowns.
- Built a GitHub-ready delivery workflow with deterministic seed data, a safe SQL explorer, pytest validation, GitHub Actions CI, Streamlit Community Cloud deployment, and a GitHub Pages project overview.

## Interview explanation
**Problem:** Sales data existed as tables and static charts, but the workflow was difficult to run outside a local SQL Server environment.

**Action:** I separated the product dimension, created a portable SQLite build, introduced reusable analytical views and a Python query layer, converted static visualizations into an interactive app, and added automated quality checks and deployment documentation.

**Result:** The project can now be cloned, tested, demonstrated locally, and deployed publicly with no database credentials.
